## 关键事实

- Vercel Changelog 的标题是 `Introducing skills, the open agent skills ecosystem`。
- 页面直接强调用户可以 `discover new skills to enhance your agents`。
- 页面明确提到可以 `track usage stats and installs across the ecosystem`。
- 页面内容与站点入口都同时提到 `claude-code`、`codex`、`cursor`、`github-copilot` 等对象，说明它不是单 agent 的私有配套。

## 与本研究的关系

- 对 `03` 而言，这份材料把“现成 skill 容易找”从主观感受变成了产品层事实。
- 它也解释了为什么借鉴别人 skill 会越来越高杠杆: 当目录、统计和统一安装方式出现后，学习成本和搜索成本都会下降。
- 对最终采用建议来说，这类生态层基础设施的出现，意味着推荐结论不能只比较 skill 内容，还要比较谁在分发和可见性上占了位置。
