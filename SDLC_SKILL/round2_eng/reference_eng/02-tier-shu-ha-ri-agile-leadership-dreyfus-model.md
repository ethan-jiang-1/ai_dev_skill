## 关键事实

- PDF 使用 Shu-Ha-Ri 框架解释学习阶段：
  - Shu：遵循既定实践（follow）
  - Ha：打破/适应（break away / adapt）
  - Ri：超越并创造自己的实践（transcend / create）
- PDF 还包含对 Dreyfus model 的简要总结（以 slide bullet 形式呈现），用于描述从 novice 到 expert 的认知变化。

## 与本研究的关系

- 对 02-tier：
  - 可把 skill 难度分层与训练策略绑定：低层级更适合强约束模板与检查清单（Shu/novice），高层级更需要允许变体与反思（Ha/Ri）。
- 对 04-path：
  - 团队推广 skill 体系时，可以用 Shu-Ha-Ri 语言解释“为什么不同人需要不同的 skill 形态与约束强度”，降低抵触。
