## 关键事实

- 论文提出 ICAP 框架，将学习者的认知参与（cognitive engagement）按可观察行为分成四类模式：
  - Interactive
  - Constructive
  - Active
  - Passive
- ICAP 假说：从 passive → active → constructive → interactive，学习效果会逐步提升；论文给出机制解释并回顾实验/课堂研究中的验证证据（包含 note taking、concept mapping、self-explaining 等活动）。

## 与本研究的关系

- 为“高质量 Skill 会抵抗认知卸载”提供可操作的标准：
  - Skill 是否把用户留在 Passive（只接受答案）？
  - 还是至少推动到 Active（操作/重现）或 Constructive（生成解释/推导）甚至 Interactive（与他人/系统互动建构）？
- 可把“逆向学习（reverse learning）”落到 ICAP 上：
  - 逆向学习的关键不只是看结果，而是通过 self-explanation / critique 把使用过程推到 constructive/interactive。
