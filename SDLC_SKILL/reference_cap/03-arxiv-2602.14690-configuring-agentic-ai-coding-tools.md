## 关键事实

- 论文对多种 agentic coding tools 的仓库级配置机制做系统分析，并给出“八类机制”分类（Context Files/Settings/Skills/Subagents/Commands/Hooks/Rules/MCP）。
- 摘要宣称其覆盖 Claude Code、GitHub Copilot、Cursor、Gemini、Codex，并对 2,923 个 GitHub 仓库做采用分析。
- 摘要层面的主要发现包括：
  - Context Files 主导，常常是仓库唯一配置机制
  - AGENTS.md 作为跨工具互操作标准正在出现
  - Skills/Subagents 的采用整体浅，且多数技能以静态指令为主，而非可执行工作流
  - 不同工具形成不同配置文化（Claude Code 使用更广机制谱）
- Table 1 给出不同工具对上述机制的典型落盘位置（如 `.claude/skills/`、`.github/skills/`、`.codex/skills/` 等）。
