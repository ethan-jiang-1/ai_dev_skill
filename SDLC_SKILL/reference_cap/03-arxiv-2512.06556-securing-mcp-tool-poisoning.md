## 关键事实

- 摘要定义的三类攻击：
  - Tool Poisoning：在 tool descriptors 中隐藏对抗指令
  - Shadowing：通过污染共享上下文间接影响“可信工具”的行为
  - Rug Pulls：在审批后修改 descriptors 以改变行为。`source_url`
- 摘要提出的三层防御：
  - RSA-based manifest signing（descriptor integrity）
  - LLM-on-LLM semantic vetting（检测可疑 tool 定义）
  - lightweight heuristic runtime guardrails（阻断异常行为）。`source_url`
- 摘要报告的量化结果（高层）：GPT-4 可阻断约 71% 的 unsafe tool calls；DeepSeek 与 Llama-3.5 在不同攻击类型上表现不同，体现安全-延迟权衡。`source_url`
