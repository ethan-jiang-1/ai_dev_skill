## Key Facts

- Agent Skills 是一套开放格式：以“文件夹”打包指令、脚本与资源，供 agents 发现并按需使用。
- Skills 的目标是补齐 agents 做“真实工作”时缺失的过程化知识与组织/团队/个人上下文，并支持按需加载。
- 对 skill 作者：强调“一次构建，多产品部署”的可移植性。
- 对兼容 agent：技能支持能让终端用户“开箱即用”赋予新能力。
- 对团队/企业：强调把组织知识固化为可移植、可版本控制的包。
- 文档明确该格式最初由 Anthropic 开发并以开放标准形式发布，支持社区贡献（GitHub/Discord）。

## Claims Supported

- “skill 生态的关键不是单段 prompt，而是可版本控制、可移植、可按需加载的能力包。”（支撑 digested_src/主题1 的‘标准层事实’）
