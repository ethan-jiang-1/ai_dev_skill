## 关键事实

- The current Codex use-cases page explicitly lists `Save workflows as skills`.
- The same page also presents Codex use cases for:
  - understanding large codebases
  - reviewing pull requests faster
  - building responsive front-end designs
  - automating bug triage
  - iterating on difficult problems
  - generating slide decks
- This indicates skills are positioned as a practical workflow-retention layer, not just a power-user config surface.
