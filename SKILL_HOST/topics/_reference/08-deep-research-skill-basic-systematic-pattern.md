## 关键事实

- The skill frames deep research as a systematic process:
  - clarify the question
  - identify key aspects
  - gather sources
  - evaluate credibility
  - synthesize findings
- It explicitly emphasizes multi-perspective analysis and source quality evaluation.

## 与本研究的关系

- Useful for Topic `08` as the baseline “structured research workflow” pattern.

## 可直接引用的术语 / 概念

- `clarify the research question`
- `identify key aspects`
- `evaluate the quality and credibility of sources`
- `synthesize information from multiple perspectives`

## 模型 / 宿主 / 版本相关信息

- This type of skill is relatively portable because it encodes workflow, not only tool-specific integration.

## 风险与局限

- Without stronger tool/runtime support, this pattern can remain a better checklist rather than a truly deeper execution engine.
