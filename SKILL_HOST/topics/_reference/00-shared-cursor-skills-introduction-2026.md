## 关键事实

- On `2026-01-22`, Cursor announced support for Agent Skills in both the editor and CLI.
- Cursor says agents can discover and apply skills when domain-specific knowledge and workflows are relevant.
- Cursor also supports invoking a skill via the slash-command menu.
- Cursor positions skills as better than always-on declarative rules for dynamic context discovery and procedural how-to instructions.
- The same release also introduced subagents, linking skills to a broader parallel-agent story.
