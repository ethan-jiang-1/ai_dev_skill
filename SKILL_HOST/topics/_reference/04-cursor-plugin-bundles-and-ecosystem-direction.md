## 关键事实

- Cursor changelog states marketplace plugins can package:
  - skills
  - subagents
  - MCP servers
  - hooks
  - rules
- This turns skills into one component inside a bundled extension supply chain.
- It also implies installation, trust, and lifecycle may increasingly happen at plugin level rather than file-copy level.

## 与本研究的关系

- Important for Topic `04` because this changes how users may increasingly encounter and maintain skills.
- Important for Topic `06` because bundle-based distribution is a major comparison axis against Claude, Codex, and OpenCode.
