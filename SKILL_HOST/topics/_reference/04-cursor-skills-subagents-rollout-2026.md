- Cursor 2.4 introduced Agent Skills in both editor and CLI.
- Cursor describes skills as an open standard for packaging reusable knowledge and scripts.
- Skills can be invoked manually from the slash-command menu.
- Cursor positions skills as a way to avoid always-on rules for procedural or dynamic workflows.
- The same 2.4 release introduced subagents.
